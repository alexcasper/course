  Summary & Looking Ahead

 

You have seen how loops can be employed in SQL and have been introduced to the Oracle extension PL/SQL (“Procedural Language SQL”).

You have also briefly seen what collection types can be employed, with focus on arrays.

You have briefly learnt about the different types of triggers that can be established. You have seen a detailed example of an “AFTER” trigger and a brief example of a “BEFORE” trigger, as well as the advantages and disadvantages of triggers.

You have been introduced to the importance of managing transactions, and the key functionalities of transaction support.  

Learning Objectives

 

You should now be able to:

  • Understand the basic principles of relational databases: their structure, use and how to interrogate them. Basic SQL syntax and query structure.
    • Describe loops and arrays.
    • Explain how to use loops to access arrays.
    • Explain the function of Triggers.
    • Describe Transactions and Transaction Management.
    • Explain the database systems development lifecycle.
  • Code structurally correct SQL commands. Perform basic data manipulation and information retrieval operations using SQL.
    • Perform coding in SQL.
      • Understand logical arguments, identifying the assumptions made and the conclusions drawn .
      • Demonstrate the ability to analyse, evaluate and correctly interpret data.

 

If you are not able to confidently discuss the above, you should revisit the material and perform self-directed research.
