User Interfaces and ServiceNow

Databases traditionally were hard to make sense of - they required knowledge of complex query languages, and often presented data in plain text. While databases were meant to be a central repository for members of an organisation to store and access information, the lack of a user interface provided a blocker for many to do so. As such, people would choose not to use an organisational database and use other means to record or retain data - spreadsheets, text files or even paper!

ServiceNow differentiates itself from other databases by providing a set of user interfaces to make storing and managing data much easier. There is a Service Portal, designed for end users, that will allow them to access information, record problems, and order items or services. The Now Platform, while designed for people 'behind the scenes' of a business, is also easy to access. While it takes some time to become familiar with all of its elements, the platform will allow you to collect, organise and present data without using code.
