Welcome to the optional module! Here we will focus on the data management methods by using Tableau. Tableau is an amazing platform for exploring, understanding and taking decisions based on your data. We will learn how to carry out data discovery methods and techniques for analysis and storytelling. 

Tableau is a very popular option for developing data analytics without the need to write complex scripts, instead, you will interact with a platform and by using the graphical user interface you will create visualisations. One of the biggest advantages of Tableau is that we see results immediately, as we are working in real-time. That means we can apply different analysis and visualisation methods and discover the best one to use.
