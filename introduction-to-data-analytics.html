Introduction to Data Analytics

Overview

The exponential growth of big data, supported by increasingly sophisticated algorithms and enhanced computing power, has brought about the age of analytics. We now live in a data-driven world, and there is a strong demand for professionals with data skills across nearly all industries.

In this session, we’ll introduce the components of data analytics and the tools we use to work with data. We’ll also see how to leverage visual aids and storytelling techniques to communicate the insights we gain from the harnessed data.
