In this module, we focused on data management and fundamental visualisations/decision-making methods by using Tableau. We used Tableau, that is a popular data management platform for exploring, understanding and taking decisions based on your data. We learned how to carry out data discovery methods and techniques for analysis and storytelling.

  • We explored data analysis pipelines with Tableau by following a tutorial.
  • We created data analysis solutions, and we learned how to create dashboards and visualisations in Tableau
  • We developed data decision-making strategies with Tableau
