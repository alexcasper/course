Activity: Accessing NowLearning and your next course (12.0: Flow Designer Overview)

You now need to access your NowLearning account and continue with the ServiceNow Fundamentals Path. See below for a reminder on how to access NowLearning:

  1. Go to NowLearning from your Bookmark or using this link: https://nowlearning.service-now.com/lxp (Links to an external site.) 
  2. Login to your account using the email address you provided at registration; you will be emailed a 6-digit passcode each time you Login to NowLearning
  3. From ‘My Learning Plan’, or from the Fundamentals Path, select Course 12.0: Flow Designer Overview 

Complete Course 12.0: Flow Designer Overview to learn how to set up automated processes in order to assist in the completion of daily tasks, notifications, approvals and record operations.

This course contains lessons, including links to additional information and resources, and a knowledge check.

During the course, you may want to follow along / copy some of the steps using your personal ServiceNow instance (this is optional). You won't need to validate any tasks but it would start to familiarise you with using the tool.

Come back here when you are finished.
