ITSM Supporting Applications

The following video looks at some of the applications from the ServiceNow platform that can be used in conjunction with ITSM applications above.

The configuration management database (CMDB) is one of the critically important supporting applications; while not technically a part of service management, it stores all of the configuration information for hardware, software and abstract services across the organisation. It allows for dependencies between different pieces of running equipment to be tracked. A distinction needs to be made between asset management and configuration management - while asset management deals with items (including intangible assets such as software), configuration management tracks how these items are being used to support business processes.

Alongside the CMDB is the Common Service Data Model - a data model associated with the CMDB used across all ServiceNow products. While you may be familiar with a conceptual data model, the CSDM is at a higher level, in that it is not attempting to reflect every single table within the database, but rather acting as a guide showing five domains, used by different parts of the business, and how these domains fit together.

Foundation

Foundation

A shared schema used across the business, representing things like users, groups, locations and contracts.

Build

Build

Used to gain visibility of software applications and infrastructure in development, represented by components.

Design

Design

Used to represent the technology or data used to meet a particular business purpose - the word application here should not be understood as a software application, but rather the way a piece of technology is applied.

Manage Technical Services 

Manage Technical Services

Tables represent the technical services - services that can potentially be provided - and technical service offerings - services configured to meet certain geographic, financial or resource requirements. An application service is an actual deployed technical service, running on particular piece of technology. In addition, it includes tables representing the hardware, software, and abstract 'applications' needed to facilitate those application services - the CMDB.

Sell / Deliver

Sell/Deliver

A business service is a service that provides some value, either for an external customer or an external server. Often, it will depend on technology - using one or more technology service offerings implemented as application services.
