Tasks

Tasks are how work is handled in ServiceNow. Tasks can be incidents, problems, change requests etc. The next course (7.0 Task Management Overview) reviews task management including the use of roles, groups and user assignments. Visual Task Boards are also introduced. This course has 2 Activities.

There is a lot of reading in this course so make sure you continue to scroll down through all of the lessons.

There are options to click on the + sign to learn more about certain functionality. There are also ‘interactive’ parts where you click an icon to see extra detail; make sure you take the time to do this.
