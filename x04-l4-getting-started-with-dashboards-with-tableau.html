Getting started with Dashboards with Tableau

The final part of our tutorial focuses on the creation of dashboards with Tableau. A dashboard is a collection of several views, letting you compare a variety of data simultaneously. For example, if you have a set of views that you review every day, you can create a dashboard that displays all the views at once, rather than navigate to separate worksheets.

In this tutorial, we will follow the next steps:

Step 8: We will get introduced to Tableau dashboards.

Step 9: Explore the Tableau views to build a dashboard.

Step 10: Use Tableau to create advanced dashboard features and dashboard objects.

Step 11: Explore methods to format professional dashboards.

Step 12: Explore methods to create interactive dashboards.

In this module, we will follow the official Tableau tutorials to learn the basics.
