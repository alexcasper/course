 UI Design

Developing User Interfaces

Not every information system will have an easy to use user interface. For many applications, functionality is the main concern, and users will need to put the time in to learn the specific interface of the system. For many applications, however, interface design will determine the ultimate success of the system. As a result, design and the user experience is often seen as critical for ensuring system success. This has lead to the growth of roles such as the design-focused User Interface Designer and the more research-oriented User Experience Designer.

For platforms, a regular selling point is that they make it easier for users across the organisation to produce effective applications - this is the motivation behind the low-code movement.

UI Builder Instructions

You should be able to find it in your instance by selecting 'all', typing UI builder and then selecting the link. 

Select 'Create experience' on the next page, on the top right:

Select any name you want. It is recommended to choose agent workspace as your shell. Then select create and then open experience.

Select 'Create a Page'.

Use any name, and do not select a template.

Close the window that comes up that says 'Success!'

Select add component under content, type button, and then select a button to add it to the page.

Change the style to align the button to the centre of the page. 

You may also wish to add padding (click the +):

We are now going to add an alert. Next to content, select +Add and choose modal, then select alert.
