  Summary & Looking Ahead

 

This module introduced you to Structured Query Language (SQL) the most widely employed database query language. SQL is an industry standard across a large part of the globe. It is a fourth generation language (4GL) and uses common English words as commands.

You were also introduced to the SQLite3 and Oracle Live SQL applications. You have also seen how to employ six of the common and important SQL commands: SELECT, FROM and WHERE, ORDER BY, GROUP BY and HAVING. In addition, you have also understood how to perform arithmetic operations, and set up constraints such as Primary and Foreign Keys.

Next, you will look at more operations that can be performed on tables as well as querying more than one table in a single SQL statement.

Learning Objectives

You should now be able to:

  • Understand the basic principles of relational databases: their structure, use and how to interrogate them. Design and implement basic relational databases.
    • Explain the importance of SQL for accessing, processing and updating the contents of a relational database.
  • Basic SQL syntax and query structure. Demonstrate the ability to analyse, evaluate and correctly interpret data.
    • Detail the definitions, manipulations, functions, keys and clauses in SQL with illustrative examples.
  • Code structurally correct SQL commands. Perform basic data manipulation and information retrieval operations using SQL.
    • Perform coding in SQL.

 

If you are not able to confidently discuss the above, you should revisit the material and perform self-directed research.
