  Summary & Looking Ahead

 

You have seen how to work with other database objects namely sequences, indexes and views.

The sequences can keep track of the unique values of specific items within a database. You have also seen that using indexes can increase the efficiency of information retrieval from the database. You have also learnt about views, and how important they are as database objects. Views can be seen as “virtual tables”.

You have learnt about the different types of single-row and multiple-row functions that can be employed in SQL including character manipulation and regular expressions.

You have also seen additional statistical functions and you have seen which of the described functions are available in SQLite3 and Oracle Live SQL.

Next, you will be introduced to some advanced techniques such as loops, triggers and transactions. These are especially important when there are multiple users accessing the database at the same time - how to avoid mishaps is of paramount importance!

Learning Objectives

 You should now be able to:

  • Demonstrate skill in numeracy and computation.
  • Understand the basic principles of relational databases: their structure, use and how to interrogate them. Understand logical arguments, identifying the assumptions made and the conclusions drawn. Demonstrate the ability to analyse, evaluate and correctly interpret data.
    • Explain sequences, indexes, views, selected row functions and additional aggregate functions.
    • Explain the purpose of UNIQUE and CHECK.
  • Basic SQL syntax and query structure. Code structurally correct SQL commands. Design and implement basic relational databases.
    • Perform coding in SQL.

If you are not able to confidently discuss the above, you should revisit the material and perform self-directed research.
