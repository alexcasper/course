 Overview

 

We have touched upon sequences, indexes, views, selected row functions, and additional aggregate functions. 

In this Module, we’ll look at loops and arrays and learn how to use loops to access arrays. The module will explain how to establish and drop triggers and touch upon their advantages and disadvantages. Finally we’ll look at what a transaction is, and will see what constitutes the database development life cycle.

 

 Learning Objectives

By the end of this module, you should be able to:

  • Understand the basic principles of relational databases: their structure, use and how to interrogate them. Basic SQL syntax and query structure.
    • Describe loops and arrays.
    • Explain how to use loops to access arrays.
    • Explain the function of Triggers.
    • Describe Transactions.
    • The database development life cycle.
  • Code structurally correct SQL commands. Perform basic data manipulation and information retrieval operations using SQL.
    • Perform coding in SQL.
  • Understand logical arguments, identifying the assumptions made and the conclusions drawn.
  • Demonstrate the ability to analyse, evaluate and correctly interpret data.
