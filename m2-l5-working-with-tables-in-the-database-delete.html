Working with Tables in the Database

 

We can create tables by using the ‘CREATE TABLE’ statement. We can specify the attribute names along with their domains (datatypes) and then populate them.

The general statement for creating a table is as follows:

CREATE TABLE table_name (
Column_1 datatype,
Column_2 datatype,
Column_3 datatype,
Column_4 datatype,…
);

 When we create a table, we can also specify additional parameters for the attributes (columns) such as NOT NULL, PRIMARY KEY and FOREIGN KEY to give these attributes specific behaviours. In addition to creating and populating tables, you can also delete tables. 

 Finally, to exit SQLite3, simply type in .quit or .exit at the prompt. The database will be automatically saved.
