Database Design and Management

 

Course Description

This course is designed to familiarise you with the underlying principles and concepts behind relational databases.

You will learn how to proceed in designing a relational database.

You will then use the reporting language SQL to interrogate relational databases.

This will involve creating and accessing data tables, indexing, performing arithmetic operations, loops, arrays, multiple table processing, I/O operations, data type conversions and views.

In addition, you will have the opportunity to apply your knowledge of relational databases using an industry standard cloud-based technology e.g. ServiceNow.

Course Outcomes

KNOWLEDGE & UNDERSTANDING

K1. Understand the basic principles of relational databases: their structure, use and how to interrogate them.

K2. Basic SQL syntax and query structure.

K3. Have knowledge of the regulations, ethical and security considerations required in modern data storage.

SUBJECT SPECIFIC SKILLS

S1. Code structurally correct SQL commands.

S2. Design and implement basic relational databases.

S3. Perform basic data manipulation and information retrieval operations using SQL.

TRANSFERABLE & PROFESSIONAL SKILLS

T1. Demonstrate skill in numeracy and computation.

T1. Display a developing technical proficiency of written English skills that demonstrates an ability to communicate clearly and accurately when producing structured and coherent pieces of text.

T2. Understand logical arguments, identifying the assumptions made and the conclusions drawn.

T3. Demonstrate the ability to analyse, evaluate and correctly interpret data.
