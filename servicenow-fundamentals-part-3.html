ServiceNow Fundamentals - Part 3

 

By now, you will have seen ServiceNow in action which will help set the context for the ServiceNow Fundamentals courses.

Most courses within ServiceNow Fundamentals Path include short ‘Digitize it’ videos where ServiceNow team members will introduce you to high level content with light banter.

Tips

  • Ensure you watch all the core videos as this is how you get the green circle completed against each course
  • Make sure you scroll to the bottom of all pages as there are often multiple videos / lessons
  • Once you have completed a course, you will see a message / notification stating you have unlocked a new badge. It will then move you automatically to the next course
