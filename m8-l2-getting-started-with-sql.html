 Getting Started with SQL

 

For the majority of this course you will be making use of a database application program called SQLite3 – which is like a “light version” of a major SQL application program (e.g. Oracle or Microsoft SQL).

The SQLite3 application program understands the majority of the SQL commands.

The SQLite3 application program also permits execution of tasks specific to the program, for example, importing a data file. 

However, should there be situations when SQLite3 appears unable to execute the SQL commands as advised, you can also use Oracle Live SQL. This uses the Oracle Database 19c Enterprise Edition (19.8.0.0.0) and Oracle’s Application Express (APEX). You will need to create a user account for yourself in order to avail of the platform. Later in this module, you will be introduced to Oracle Live SQL. 

You will be advised about which platform will be suited for any challenge-based exercises you will undertake later in the module, and throughout the remainder of the course.

 

Installing SQLite3 (for Windows 7, 8, 8.1 or 10)

Next, let's find out how you can install the SQLite3 application program onto your PC/laptop.

The process of installing the SQLite3 application program involves downloading a Zip file containing the program, extracting the three files contained within the Zip file, and placing them in a preferred location on the computer.

No additional system files, or changes to current system files, are required. This means that even if you do not have administrative privileges on your work PC/laptop running a Windows operating system, you should still be able to “install” the program in your user space (e.g. "C:\Users\Name\"). If you are unsure, please consult with your IT department.

The video also illustrates the use of a terminal in order to use the SQLite3 program. Here is a reminder on how to launch the terminals in the three operating systems (please click on the appropriate tab):

Windows

Click on the Windows Start menu (usually bottom left of screen) and type cmd in the Search Program field and then press enter.

The MS-DOS Command console should launch.

Linux (Ubuntu)

You can launch a terminal by using the following key combination: CTRL-ALT-T which means to press down on the CTRL, ALT and T keys at the same time.

A new terminal should appear as shown in the above video.

Mac

The terminal can normally be launched by clicking on the Terminal icon in the Dock (the ribbon of icons usually located at the bottom of the screen).

Alternatively, you can use the Launchpad (the "rocket" icon), and select Terminal from the display of icons.
