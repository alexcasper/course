Additional ServiceNow Content for System Administrators and Developers (Course 15.0)

Course 15.0 is the final ServiceNow course that you will cover in Database Design & Management - I.

The course is split into 10 modules, most of which contain lessons, knowledge checks and activities that you can undertake in your personal ServiceNow Instance.

Remember, there will be no need for validation when using your personal ServiceNow Instance.

The 10 modules are summarised in the table below:

Module Title Description
15.1 Data Schema Overview Talks about database tables and fields, and common relationships, for example one-to-many.
15.2 Access Control Overview Talks about restricting access to records and applications via the setting of permissions of users, groups and roles.
15.3 Import Data Overview Talks about importing datasets of different sources and formats such as Excel, CSV and XML, into the ServiceNow platform. Transforming and coalescing are also discussed.
15.4 Configuration Management Database (CMDB) Overview Talks about configuration items that are controlled by a company to ensure services are managed and delivered such as business applications and hardware. Also introduces the Common Service Data Model (CSDM).
15.5 Catalog Administration Overview Talks about the service catalog and how to ensure that orders and inventory are efficiently tracked.
15.6 Platform Scripting Overview Talks about creating scripts for specific and more focused tasks leading to improved customisation. JavaScript tends to be the language employed.
15.7 Migration and Integration Overview Talks about migrating and integrating configurations from one instance into another.
15.8 Custom Application Development Overview Talks about how to build custom applications via the 'Guided Application Creator'.
15.9 Events Overview Talks about the creations of logs to capture events such as a user logging in to the system, or a request being approved.
15.10 Instance Stats Overview Talks about the statistics that is used to monitor system performance and diagnostics.
